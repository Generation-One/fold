This is comprehensive documentation for the Fold semantic memory system, a holographic memory platform designed for development teams to store and retrieve semantic knowledge using vector embeddings. It covers system architecture, installation procedures, configuration requirements, API usage, and troubleshooting guidance. The documentation describes Fold's core components including an Axum-based REST API, SQLite metadata storage, Qdrant vector database integration, and multi-provider LLM services with fallback mechanisms.
