This file implements a user management and authentication system with service-oriented architecture. It defines a User dataclass with role-based access control (ADMIN, EDITOR, VIEWER, GUEST), a UserService for CRUD operations on users, and an AuthService for token-based authentication. The module demonstrates clean separation of concerns with dedicated services handling distinct responsibilities, type hints for type safety, and includes a partial password hashing utility function.
